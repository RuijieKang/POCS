Bring up and down the kafka environment

- docker-compose up -d
- docker-compose down

Manage the kafka cluster

- Kafka Manager console http://localhost:9000/

Create topics used by the application
- random-number

Terms:
    Topics : logical table
    Partition: A segment of data in topics
    Boker: physical node
